### MLQA_BERT:

* How to download MLQA dataset:

```Bash
pip install gdown
sudo apt-get install unzip
gdown https://dl.fbaipublicfiles.com/MLQA/MLQA_V1.zip
unzip MLQA_V1.zip
rm MLQA_V1.zip
```

* Python packages to install:

```Bash
pip install transformers
```
